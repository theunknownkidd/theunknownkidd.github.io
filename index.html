Assignment 3 Working With Images

What Is A PNG Best For

PNG is an excellent file format for wireframes where there’s a lot of text and straight lines. They also work great for screenshots. This image format retains its high quality even when compressed, and works for both complicated and simple images

What Is A JPEG Best For

JPG files are widely used because the compression algorithm significantly reduces the size of the file, which makes it ideal for sharing, storing, and displaying on websites. However, this JPG compression also reduces the quality of the image, which might be noticeable if it's highly compressed.

What Is A GIF Best For

In its simplest form, a GIF (pronounced “gif” or “jiff”) is just an image file. Like the JPEG or PNG file formats, the GIF format can be used to make still images. But the GIF format has a special feature—it can also be used to create animated images
